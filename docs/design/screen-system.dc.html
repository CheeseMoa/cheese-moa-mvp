사용 규칙
① 토큰 원천은 tailwind.config.js 하나 — 이 문서의 값은 그 사본이며, 충돌 시 코드가 이긴다 (이 문서를 갱신할 것).  ② Jua는 로고·헤딩·EmptyState 타이틀에만, 본문·버튼·배지는 Gothic A1.  ③ 치즈 옐로우 위 흰 글자 금지 — primary 버튼 텍스트는 text(#3A3128).  ④ AlbumCard 테두리 규칙 고정 — 갈색 실선=검토완료 · 회색 점선=미검토.  ⑤ 특수 앨범(분류가 어려워요·눈감음·흔들림)은 뷰어 비노출.  ⑥ 미확정(screen-spec §5): 홈 관리자 배지 · 품질 제외 사진 라우팅 · common 앨범 정의 — 임의 구현 금지.
